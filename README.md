# Simple Recommendation System

## Overview

This project is a basic recommendation system developed in Python. The system takes a user's interest as input and recommends relevant items based on predefined categories.

The project demonstrates the fundamental concepts behind recommendation systems, including user preference matching and recommendation generation.

## Goal

Create a simple recommendation system based on user preferences.

## Objectives

- Take user input (choices or interests)
- Match user preferences using logical conditions
- Display relevant recommendations
- Demonstrate basic recommendation system concepts

## Features

- Accepts user interests as input
- Matches interests with predefined categories
- Provides personalized recommendations
- Handles unknown interests gracefully

## Technologies Used

- Python

## How It Works

1. The user enters an interest category.
2. The system searches for that category in its recommendation database.
3. If a match is found, relevant recommendations are displayed.
4. If no match is found, an appropriate message is shown.

## Available Categories

- Movies
- Books
- Sports
- Music
- Technology

## Sample Input

```text
Enter your interest: movies
```

## Sample Output

```text
Recommended Items for You:
- Inception
- Interstellar
- The Dark Knight
```

1. Save the code as:
